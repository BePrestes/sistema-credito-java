# Sistema de Crédito em Java

Projeto desenvolvido para praticar Java e Programação Orientada a Objetos,
simulando um sistema de análise de crédito via terminal.

## Funcionalidades
> Cadastro de clientes,
> Listagem de clientes,
> Avaliação de crédito via regras,
> Mensagens de interação (Bem-vindo, Aguarde, Volte sempre).

## Conceitos aplicados
> POO (Programação Orientada a Objetos),
> Camadas (Controller, Service, Repository),
> Regra de negócio desacoplada (Motor de Regras),
> Enum para status,
> Git e GitHub.
